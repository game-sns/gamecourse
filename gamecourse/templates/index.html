GAlaxy Machine learning for Emission lines

GAME is a Machine Learning code that infers the ISM physical properties by analyzing the emission-line intensities in a galaxy spectrum (for the complete description see Ucci et al. 2017 and Ucci et al. 2018). The code is based on a Supervised Machine Learning algorithm, and it is trained with a large library of synthetica spectra spanning many different ISM phases, including HII (ionized) regions, PDRs, and neutral regions.

Given a set of input lines in a spectrum, GAME performs a training on the library, then evaluates the line intensities to give a determination of the physical properties. The errors on the input emission-line intensities and the uncertainties on the physical properties determinations are also taken into account (Ucci et al. 2018).

The physical properties that GAME is able to infer are: gas density (n), column density (NH), far-ultraviolet (FUV, 6–13.6 eV) flux (G), ionization parameter (U), metallicity (Z), escape fraction (fesc), visual extinction (AV).
